# Введение

**Цель работы:**  
Исследовать возможности на выбор одного из десктопных клиентов для работы с Git при выполнении наиболее распространенных команд.

**Задачи:**
1. Основная задача выполнить основные операции: `init`, `touch`, `log`, `status`, `push`, `pull`, `add`, `commit`, `diff`, `reset`, `checkout`, `branch`, `merge`, `rm`.
2. Инициализировать локальный репозиторий и связать его с удалённым.
3. Настроить аутентификацию через Personal Access Token (PAT).
4. Создать и удалить ветку, провести слияние.
5. Оформить отчёт в Markdown и загрузить его в репозиторий-портфолио.
